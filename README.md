# 🏨 AtliQ Grands Hospitality Analysis \| Power BI Project

## 📌 Overview

This project delivers a comprehensive business intelligence solution for
AtliQ Grands, a chain of luxury hotels in India. Built using Power BI,
the dashboard analyzes performance across revenue, occupancy, ratings,
and bookings.

## 🎯 Business Problem

-   Declining market share\
-   High cancellation rates\
-   Lack of data-driven insights

## 📊 Key Metrics

-   Revenue: 499M\
-   Occupancy: 57.45%\
-   Avg Rating: 3.62\
-   Cancellation Rate: 24.4%

## 🧩 Data Model

-   Fact Tables: fact_bookings, fact_aggregated_bookings\
-   Dimension Tables: dim_date, dim_hotels, dim_rooms

## 🛠️ Tools Used

-   Power BI\
-   Power Query\
-   DAX\
-   Excel

## 🚀 Insights

-   Mumbai drives highest revenue\
-   Delhi has best ratings\
-   High cancellations impact revenue\
-   Demand-supply gap exists
